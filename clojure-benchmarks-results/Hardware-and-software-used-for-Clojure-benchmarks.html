Mac OS X 10.6.8

All Linux runs were made running Ubuntu 12.04.1 server 64-bit as a virtual machine with VirtualBox 4.2.4 (TBD: get more version details), all on top of Mac OS X 10.6.8.  The virtual machine was configured with 4 virtual processors and TBD GBytes of RAM.  The virtual hard drive was created by pre-allocating all of its capacity at creation time.  The option to create a dynamically growing virtual disk file was not used.  VirtualBox gave reason to believe that the pre-allocation of virtual disk should lead to better performance for file I/O.  The graphs show the Linux kernel version number.
